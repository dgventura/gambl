Game_Music_Emu 0.2.6: Multi-Format Game Music Emulators
-------------------------------------------------------
Game_Music_Emu is a collection of portable video game music emulators for the
following file formats: Game Boy GBS, Sega Master System/Genesis VGM, Sega
Genesis GYM, Super Nintendo SPC, and Nintendo NSF (with VRC6, Namco 106, and
FME-7 expansion sound).

Getting Started
---------------
Build a program consisting of the included source files (except
demo_effects.cpp). Be sure "test.nsf" is in the same directory. Running the
program should generate a WAVE sound file "out.wav" of music.

For a full example of using Game_Music_Emu in a music player, see the Game
Music Box source code: http://www.slack.net/~ant/game-music-box/dev.html

See notes.txt for more information, and respective header (.h) files for
reference. Visit the discussion forum to get assistance.
